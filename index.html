CSS stands for Cascading Style Sheets and is a styling sheet language used for designing the content of a webpage. CSS can be used three ways which are internally, externally, and inline. Inline css means that you have change the design of something in an element. For example, < h1 style="background-color: #00BBF9;">Thomas A. Edison CTE High School This would make the background color of a level 1 heading, deep sky blue. Internal CSS means you're still working within the HTML file. Internal CSS has to be done within the head tags because it is data for the webpage. External CSS means you are outside of the HTML document. In order for the changes in the external CSS to show, you would have to use the link tag like so: < link type="text/css" rel="stylesheet" href="mkpro.css">. These are the three ways to use CSS in a HTML file.
